# Skill: Project Management

## Descripción

Skill para gestionar proyectos usando metodologías ágiles (Scrum/Kanban).

## Triggers

- "gestionar proyecto"
- "project management"
- "敏捷"
- "scrum"
- "kanban"

## Actividades

### Inicio de Proyecto
1. Definir visión del producto
2. Identificar stakeholders
3. Crear roadmap inicial
4. Definir sprint cadence (1-2 semanas)

### Durante el Proyecto
- Revisar backlog
- Actualizar prioridades
- Identificar blockers
- Facilitar ceremonias (daily standup, sprint review, retro)

### Métricas
- Velocity
- Burndown/Burnup
- Cycle time
- Lead time

## Herramientas

- **Jira/Trello/Linear** para tracking
- **Funemon** para persistir decisiones importantes
- **SDD** para análisis de features

## Output

- Roadmap actualizado
- Sprint goals definidos
- Blockers identificados
- Métricas de equipo
